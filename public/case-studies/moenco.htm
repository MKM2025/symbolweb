Unified Communications Solution for MOENCO

Client: The Motor & Engineering Company of Ethiopia Ltd. S.C (MOENCO)
Project Name: Design, Supply, and Installation of Unified Communications Solution
Location: Addis Ababa, Ethiopia

🧩 Project Background

MOENCO, Ethiopia’s premier automotive distributor, faced increasing limitations with its legacy analog telephony system. It lacked flexibility and modern collaboration features that are essential for efficient enterprise communication. In response, MOENCO engaged Symbol Technologies to design and implement a Unified Communications (UC) platform capable of supporting modern voice, video, and mobility features across its headquarters.

🎯 Project Objectives

  • Replace the outdated analog telephony infrastructure
  • Introduce enterprise-grade communication features like voicemail, auto-attendant, call pickup, conferencing, and mobility
  • Enhance collaboration and communication efficiency across departments and leadership teams

🛠️ Scope of Services Delivered

Symbol Technologies deployed a full-featured Cisco Unified Communications platform hosted on a virtualized environment. The key components of the deployment included:

Virtualized UC Infrastructure

  • Deployed Cisco Unified Communications Manager (CUCM) and Cisco Unity Connection (CUC) on VMware ESXi
  • Licensed and configured ESXi host environments
  • Configured CUCM and CUC for seamless integration with Microsoft Active Directory

Telephony Features Configuration

  • Created Voice VLANs for secure voice traffic segmentation
  • Configured directory numbers based on organizational departments
  • Set up PSTN access for external communication
  • Enabled advanced features: voicemail, visual voicemail, intercom, call pickup, auto-attendant, callback, and mobility

Executive & Departmental Enhancements

  • Personalized MOENCO logo set as background on all IP phones
  • Enabled video call capabilities for executive phones
  • Deployed Cisco Jabber soft client with full CUCM integration for mobility
  • Configured Class of Service (CoS) restrictions for user-level call privileges
  • Established conference calling via Cisco 7937 conference station
  • Performed internal group call setups for collaborative departments

System Hardening & Support

  • Configured the voice gateway as NTP server for system synchronization
  • Implemented system backup and restore procedures
  • Performed custom telephony configurations per client-specific requests

✅ Project Outcome

The new Unified Communications solution significantly transformed MOENCO’s internal and external communication capabilities. Departments now benefit from centralized management, high availability, advanced voice services, and seamless integration with business systems. Executives enjoy rich collaboration experiences with features such as video calls, visual voicemail, and mobile access via soft clients.

By upgrading to an IP-based, virtualized UC solution, MOENCO is now better equipped to support its business growth with scalable, flexible, and secure communications.
